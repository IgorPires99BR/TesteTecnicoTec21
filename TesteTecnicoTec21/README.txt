Esse projeto foi criado com o intuito de apresentar as funcionalidades 
exigidas no teste tecnico feito para a Tech 21.


Um sistema ira gerar uma importação de arquivos xml, no modelo como foi passado,
e depois disso o sistema ira enviar esses dados para o banco de dados, e permitir
que o usuário adicione os valores de Data de apresentação e Valor das custas.

Depois de salvo, o sistema envia os dados novamente para o banco.

Divirtam-se.
